# Ícones da Extensão

Por enquanto, use ícones temporários para desenvolvimento.

Para criar os ícones:
1. Crie imagens PNG nos tamanhos 16x16, 48x48 e 128x128
2. Salve como `icon16.png`, `icon48.png` e `icon128.png`
3. Coloque-os nesta pasta `/icons`

Você pode usar ferramentas online como:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Ou criar manualmente com qualquer editor de imagens.
